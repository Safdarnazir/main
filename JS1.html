There are three ways to add JavaScript to HTML pages.

Embedding code :- In this method we use script tag (in which all java script code is written) inside head tag or inside body tag

External line:- In this method we create a separate file to hold the code of JavaScript with the (.js) extension and later include it into our HTML document using the src attribute ex- script src="hello.js" or by using link attribute

Inline code:- In this method we add java script code directly without using script tag
